# چک‌لیست تست پروژه کیستی

این سند شامل چک‌لیست کامل تست تمام صفحات و عملکردهای پروژه کیستی است.

## دستورالعمل استفاده

- برای هر مورد تست، وضعیت را با ✓ (تایید شده)، ✗ (ناموفق) یا - (غیرقابل تست) علامت بزنید
- در صورت وجود مشکل، توضیحات را در ستون "یادداشت" بنویسید
- تست‌ها باید در مرورگرهای مختلف (Chrome, Firefox, Safari, Edge) انجام شوند
- تست ریسپانسیو باید در دستگاه‌های مختلف انجام شود
