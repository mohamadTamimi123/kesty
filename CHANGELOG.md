# Changelog

تمام تغییرات قابل توجه این پروژه در این فایل مستند می‌شود.

فرمت بر اساس [Keep a Changelog](https://keepachangelog.com/fa-IR/1.0.0/) است،
و این پروژه از [Semantic Versioning](https://semver.org/lang/fa/) پیروی می‌کند.

## [1.0.0] - 2024-12-19

### اضافه شده
- سیستم احراز هویت کامل با OTP و JWT
- پنل مدیریت کاربران، شهرها و دسته‌بندی‌ها
- سیستم پیام‌رسانی Real-Time با WebSocket
- مدیریت پروژه‌ها برای مشتریان و تولیدکنندگان
- سیستم توزیع خودکار درخواست‌ها به تولیدکنندگان مرتبط
- پروفایل تولیدکنندگان با نمایش نمونه کارها و نظرات
- سیستم پرداخت با زرین‌پال
- سیستم سابسکریپشن
- بازارگاه ماشین‌آلات
- مقالات آموزشی
- سیستم Changelog با تب‌های فلو سیستم و پلتفرم نیازمندی‌ها
- گزارش پیشرفت پروژه با جزئیات کامل

### بهبود یافته
- بهینه‌سازی رابط کاربری
- بهبود تجربه کاربری در صفحات مختلف
- بهبود عملکرد سیستم پیام‌رسانی Real-Time

### امنیت
- پیاده‌سازی احراز هویت امن با JWT
- استفاده از Redis برای مدیریت OTP
- اعتبارسنجی ورودی‌ها در سمت سرور و کلاینت

### مستندات
- اضافه شدن مستندات کامل API
- ایجاد CHANGELOG
- مستندات فلوهای سیستم برای QA
